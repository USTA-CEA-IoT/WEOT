Alcance

El sistema de monitoreo ambiental será diseñado bajo los principios de el Internet de las cosas, para medir los niveles de contaminación de los principales gases de efecto invernadero como lo son dióxido de Carbono (CO2) y Metano (CH4), adicionalmente se incluye la medición de temperatura y humedad. Este sistema será implementado mediante un único prototipo en el cual se verifica la detección de las variables ambientales mencionadas anteriormente.
